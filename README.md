### how to install

I strongly recommend you to work inside a virtual environment.

```
$ pip install -r requirements.txt
```



<br/>

### how to run

run server

```
$ python server.py --video {videopath}
```



<br/>

and then run client on a separate bash

```
$ python client.py
```
